# SCP Foundation Equipment & Paratech Taxonomy
### Reference for agents designing items, loadouts, and consumables

**Status:** Research reference. Not a design decision record.
**Companion to:** `2026-07-12-scp-role-taxonomy.md` (same confidence markers, same schema conventions).
**Canon posture:** Tight, not religious.
**Game context:** Multiversal. Party: combat specialist, researcher, psionic, xenobiologist.

**Primary source:** `A Semi-Comprehensive List of Foundation Technology` (SCP Wiki, last edited 19 Jun 2026). The page itself carries a disclaimer worth internalizing: *it is a resource, not a canonical list, and authors are explicitly asked not to be pestered for "using them wrong."* You have latitude.

## 0. The design pattern that matters

**Every para-quantity in the setting has a detector and an actuator.** That symmetry is the whole item system, and it falls out of canon for free:

| Quantity | Detector | Actuator (raise) | Actuator (lower) |
|---|---|---|---|
| **Hume** (reality) | Kant Counter | Scranton Reality Anchor | **Scranton-Eamon Reality Sink** |
| **Akiva** (divinity) | Akiva Counter | Akiva-Emissive Sacramental Reactor | Randall-Yossarian Akiva Nullifier |
| **Psionic field** | **Cayce Counter** | — (innate) | Naismith-Henderson Lambda Wave Sink |
| **Thaumic / aetheric** | Brennan Thaumometric Satellite | Automated Aetheric Field Modifier | M-Series Thaumatic Damper |
| **Ectoplasm** | Kardec Counter | Ectoilluminator | Cazar-Hynman Spectral Capture Device |
| **Spacetime** | Minkowski Spacetime Monitor | Einstein-Pálsdóttir Temporal Field | XACTS |
| **Narrative** | **Pickman-Sinclair Narrative Fluctuation Detector** | K-Incubator Narrative Generator | — |
| **Semiotic stability** | — | **Gat-Hayes Semantic Stabilization Device** | — |
| **Memory** | Veritants | Mnestics (W–Z) | Amnestics (A–I) |

**Take the Hume row seriously.** Canon has both an anchor *and* a sink. Raising local Humes suppresses reality-bending; **lowering them makes bending easier**. The scrapped Cimmerian Box weaponized exactly this — drain the local Hume level, then bend reality at the target.

That makes your reality meter **bidirectional, with both directions tactically useful.** That is a better mechanic than a stability bar you merely defend. Build the game on it.

## 1. Canon confidence legend

| Marker | Meaning | Agent guidance |
|---|---|---|
| **●** | On the canonical technology list or a dedicated guide page | Use the name verbatim; keep the function |
| **○** | Canon but function listed as "Unknown," or single-article | **Free to define.** Canon is handing you a name and no constraints. |
| **✦** | Invented for this project | Must be internally consistent |

**Note the `○` items.** The canonical list contains entries whose entire description is the word "Unknown" — *Thaumatological Rifles*, *Occult Pistols*, *Reverse Matter Railguns*, *Switchback Energy Grenades*, *Zoological Grenades*, *Scranton Reality Disruptor*, *Heat-Seeking Atomic Disruptor*, *Harmonic Dispersion Wand*. These are canon names with no canon behavior. **Free real estate.** Use them.

## 3. Reality manipulation — the Scranton family

Large, well-developed, and internally inconsistent. Pick the ones you need.

| Item | Function |
|---|---|
| ● **Scranton Reality Anchor (SRA)** | The standard. Protects an area from reality-bending. |
| ● **Scranton Box** | A *container* that protects its contents from reality-bending |
| ● Lang-Scranton Stabilizer (LSS) | The prototype that became all reality-anchoring tech |
| ● **Scranton-Eamon Reality Sink (SERS)** | **Inverse of an SRA.** *Lowers* Hume count in a 10 m sphere by siphoning reality inward. Reverses on deactivation. |
| ● **Ark Reality Suppressor (ARS)** | Suppresses small reality tears to baseline. **Also used to temporarily obstruct Ways.** → multiversal |
| ● Long-Ranged Reality Adjuster (LRRA) | Causes Hume changes *outside* its own range, in multiple locations, via intradimensional rifts |
| ● **Mymar-Hayes Reality Anchor** | Works by **redistribution** — siphons Humes from one place to boost another. More potent, more volatile. |
| ● Scranton-Marlowe Topology Stabilizers | Maintain dimensional stability |
| ● Doe-Grey **Unreality** Anchor | Dept. of Unreality. Function described as "poorly-understood, possibly nonsensical." *Perfect.* |
| ● House-Nicolas Theoplanar Vacuum Unit | Reality anchor powered by Tartarean Resonance Energy; vents excess as alpha radiation |
| ○ Castro-Dunlap Reality Stabilization Field | Implied distinct from SRAs. Undefined. |
| ● **YBOD-01** | Implant placed in a **reality bender's larynx**, severing voluntary motor control. Contains a miniature SRA so the bender cannot unmake it. |
| ● **Cimmerian Box** | Scrapped 1997. Lowers local Humes so the *user* can reality-bend the target. Assassination weapon. |
| ● Gat-Hayes Semantic Stabilization Device (GHSSD) | See §12. |

**The Mymar-Hayes is the interesting one.** It doesn't *make* reality, it *moves* it. Somewhere is getting thinner so you can be solid. That's a cost, a moral problem, and a level-design constraint in one device.

## 6. Memory & cognition — the consumables tier

The amnestic classification was **revised**. The current guide (`updated-amnestics-guide`) replaces the old *strength*-based A/B/C model with a **function**-based one. Older articles still use the old model; this is a known, unresolved inconsistency in canon. Use the new one.

### Amnestics

| Class | Function | Notes |
|---|---|---|
| **A** | General retrograde | Hits the 5–6 hr reconsolidation window. The workhorse. |
| **B** | Regressive retrograde | Newest memories first, working backward. ~75 mg ≈ 24 hrs. |
| **C** | Targeted retrograde | Surgical. Any memory, any point in life. **Requires neuro-imaging + transcranial stimulation — not portable.** Site-bound. |
| **D** | Progressive retrograde | *Oldest* memories first. Rarely used, high side-effect risk. Leaves implicit memory (skills) intact. |
| **E** | Ennui | Doesn't erase — *decouples the memory from emotion* so it decays naturally. Used when the anomaly **can't** be hidden and must instead be made to seem normal. |
| **F** | Fugue | Erases identity. A new one may be supplied, or left to regrow. |
| **G** | Gaslighting | Makes memories feel dreamlike and inauthentic. Best when the subject has no physical evidence. *The version targeting non-anomalous memories was banned by the Ethics Committee, then un-banned at O5 request.* |
| **H** | Anterograde | Blocks formation of new memories while in the system |
| **I** | Transient | Temporary total amnesia; wears off |

### Mnestics

| Class | Function |
|---|---|
| **W** | Perceive and retain antimemes; general memory enhancement. **The standard Antimemetics Division dose.** |
| **X** | A *failed eternal youth serum.* Restores mind and body — then the suppressed time reasserts itself all at once. "Whiplash." |
| **Y** | Perfect recall of anything learned while it's active |
| **Z** | Biochemically incapable of forgetting anything, ever again. **Invariably fatal — seizure, within hours.** |

> **Canon warning, verbatim in spirit:** combining amnestics and mnestics is not recommended.

### Other cognition drugs

| Item | Function |
|---|---|
| ● **Veritants** | Truth serum. Used to *verify* memories. |
| ● **Agnostics** | Dept. of Surrealistics. **Inverts the brain's semiotic framework**, turning your understanding of reality sideways so that systems of "logic" which are normally incoherent become usable. Necessary to comprehend anomalies that exist outside the basic assumptions of reason. |
| ● **Gnostics** | The inverse. Entrenches your baseline epistemology. Makes you *more* certain. |
| ● Hypnestics | Induce suggestibility / compel compliance |
| ● Hypnosuggestive Agents | Precursor to amnestics. Powerful. Frequently fatal. |
| ● Amnesthesiologic Conceptual Overload (ACO) | Coerce and destroy *concepts* within a mind |
| ● Scranton Invasive Amnesticization | Brain surgery. When drugs won't do it. |

**Agnostics/Gnostics are a two-way dial on the player's own perception.** One makes the incomprehensible legible at the cost of your grip on the ordinary. The other makes you certain at the cost of being able to see what's actually there. That is a mechanic.

### Two canon facts worth stealing

1. **Amnestics are often themselves anomalies.** The guide says so plainly: many are anomalous in origin or mechanism, which effectively makes them Thaumiel-class SCPs — *handed out casually to any MTF, agent, or researcher who asks.* Your party's most routine consumable is a contained anomaly nobody thinks about.
2. **The Forgotten Epidemic.** The new *non*-anomalous amnestics can be synthesized by civilians. Street amnestics are now cooked in improvised labs. Overdoses leave people vegetative. The O5 Council reviewed this, declined to act, and shipped anyway.

### Design note: the anti-psionic kit is a leash

Look at that table again. Every counter-psionic item is standard Foundation issue. If your psionic is a **Special Asset** rather than an employee, then:

- The **Cayce Counter** in the researcher's kit measures *them*.
- The **electrum helmets** in the combat specialist's locker are rated against *them*.
- The **hollow-cavity firearm** — designed for maximum cerebral damage — has exactly one plausible target in the party.
- Somewhere in a file, there is a signed authorization for a **YBOD-01**.

You do not need a betrayal mechanic. You need an inventory screen.

## 12. Three items that are about *your* game

The repo is called `foundation_vs_slop`. If I have guessed the antagonist correctly, these three are not flavor — they are the thesis.

### ● Gat-Hayes Semantic Stabilization Device (GHSSD)
> *Prevents semiotic decay and destabilization, ontologically anchoring objects to concepts and definitions to meanings.*

A device whose entire purpose is **stopping meaning from coming loose from the things it means.** If your enemy is degradation-of-sense — content that looks like content, words that no longer point at anything — the GHSSD is the counter-weapon and canon handed it to you for free. Source: SCP-6254, "OBJECT."

### ● Deepwells
Ontologically-isolated servers whose contents survive a total reality rewrite. **Your save file, in-universe.** In a game where the world can be edited beneath the player, the Deepwell is why the player still knows.

### ● Agnostics / Gnostics
A drug that makes the incoherent legible, and a drug that makes you certain. In a game about slop, *certainty is the trap.* The Gnostic is the one that gets you killed.

## 14. Schema

```yaml
# item.schema.yaml
Item:
  id: string                    # snake_case, e.g. scranton_reality_anchor
  display_name: string
  canon_confidence: enum        # CANON | CANON_UNDEFINED | INVENTED
  canon_source: string?         # URL. Required unless INVENTED.

  category: enum                # DETECTOR | REALITY | TEMPORAL | MULTIVERSAL
                                # | CONSUMABLE | MEMETIC | THAUMATURGIC
                                # | PSIONIC | ARMAMENT | GARMENT | VEHICLE
                                # | AUTOMATION | INFRASTRUCTURE

  # --- Interaction with the meter system (see §0) ---
  affects_meter: string?        # hume | akiva | psi | eve | ectoplasm
                                # | narrative | semiotic | memory
  direction: enum?              # RAISE | LOWER | MEASURE | NULLIFY
  radius_m: number?
  duration_s: number?

  # --- Play ---
  slot: enum                    # WEAPON | TOOL | IMPLANT | WORN
                                # | DEPLOYABLE | CONSUMABLE | VEHICLE
  role_affinity: [string]       # role ids from the role taxonomy
  consumable: bool
  requires_site: bool           # true = cannot be used in the field
                                # (e.g. Class-C amnestics)

  notes: string?
```

## 17. Open questions

1. **Is the Hume meter bidirectional?** (§0, §3) Canon supports it — anchor *and* sink, with an entire scrapped weapon program built on draining reality before bending it. If yes, the reality meter is not a health bar you defend; it's a dial you play.
2. **Is the psionic carrying their own countermeasures?** (§9) The anti-psionic kit is standard issue. Whether the party is *equipped against* their teammate is a one-line inventory decision with enormous narrative consequence.
3. **Are Deepwells the save system?** (§5, §12) If reality can be rewritten and the Deepwell is the only thing that survives it, then "did you make it back to a Deepwell" is your checkpoint logic *and* your horror.
4. **What is slop, mechanically?** If it's semiotic decay, the GHSSD is your counter-weapon and the Gnostic is your poison. Tell me and I'll do a dedicated pass on the antagonist's systems.
